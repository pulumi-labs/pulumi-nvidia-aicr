# coreweave-inference-java

Deploy the NVIDIA AICR Dynamo inference stack onto a CoreWeave bare-metal
H100 cluster, in Java.

for the full description, recipe-choice notes, prerequisites, and cost.

```bash
pulumi up
```

## Clean up
